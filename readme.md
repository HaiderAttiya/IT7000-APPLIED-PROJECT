**Project #17 | IT7000 – Applied Project**  
**Bahrain Polytechnic | Semester 2, Academic Year 2025-2026**

## 📋 Project Scope

- **Problem Statement**: Most home users struggle with poor Wi-Fi coverage, IP conflicts, and lack of proper network segmentation.
- **Objectives**:
  - Design a star topology home network
  - Assign proper IP addressing and subnetting
  - Configure router, switch, and end devices
  - Simulate and test connectivity in Cisco Packet Tracer
- **Boundaries**: Focus only on a **basic home network** (no cloud integration or advanced security such as VPN/firewall rules beyond basics).

---

## 💡 Innovation Idea
This project introduces a **realistic, ready-to-deploy home network blueprint** using free tools (Cisco Packet Tracer + draw.io). It includes automated IP assignment logic and a clean documentation template that any homeowner or small business can reuse.
